# AmazonClone
This is a clone for amazon website
Amazon Clone Project
This project is a front-end web development exercise focused on recreating the landing page of Amazon using HTML5 and CSS3. It demonstrates structural layout techniques and responsive design principles common in modern e-commerce interfaces.

## Project Overview
The goal of this project was to replicate the core visual elements of the Amazon homepage, including the multi-layered navigation bar, hero section, and product grid layout. This project serves as a foundational piece for practicing CSS Flexbox and positioning.

## Key Features
Navigation Header: Includes the logo, location selector, search bar with category dropdown, and account/order links.

Sub-Navigation (Panel): Features a secondary bar for site categories like "Today's Deals" and "Customer Service."

Hero Section: A visually prominent area with a contextual message about local delivery.

Product Grid (Shop Section): A responsive grid displaying various product categories (Electronics, Apparel, etc.) using stylized cards.

Comprehensive Footer: Includes a "Back to top" button, categorized link lists, and legal/copyright information.

## Technologies Used
HTML5: Used for the semantic structure of the webpage.

CSS3: Utilized for layout (Flexbox), styling, and background image management.

Font Awesome: Integrated for scalable vector icons like the search glass, location pin, and shopping cart.

## File Structure
index.html: Contains the primary markup and structure.

style.css: Contains all styling rules and layout configurations.

Images: The project references external image files (e.g., laptop.webp, iphone.webp, jeans.jpg) for the shop section cards.

## Setup Instructions
Clone the Repository: Download or clone the project files to your local machine.

Add Assets: Ensure all image files referenced in the style.css or index.html (e.g., getyour.jpg, watch.jpg) are placed in the root directory.

Launch: Open the index.html file in any modern web browser to view the project.
